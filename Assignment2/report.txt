Purpose:
The goal of this assignment was to create a Fraction data type that allows exact arithmetic calculations without using floating-point numbers.
This ensures accurate results for operations like addition, subtraction, multiplication, division, negation, and exponentiation.

Class Design:
The Fraction class stores fractions using a numerator and a denominator, both as long integers. The constructors automatically reduce fractions
by dividing both numerator and denominator by their greatest common divisor (GCD) and make sure the denominator is positive. Special cases like
Infinity, -Infinity, and NaN are handled when the denominator is zero. All arithmetic methods return new Fraction objects, leaving the original 
fractions unchanged.

Test Cases and Results:
